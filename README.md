# Chatty Local AI

A local AI chatbot built with Node.js, Express, Ollama and Gemma.

## Features

* Local AI inference
* Conversation memory
* Chat history saving
* Custom chat interface
* No paid API required

## Tech Stack

* Node.js
* Express.js
* Ollama
* Gemma 3 1B
* HTML
* CSS
* JavaScript

## Installation

npm install

node server.js

Open:

http://localhost:3000
